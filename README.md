# ALIF Website

This is our web development project for ICT final project.  
It includes a homepage, menu page, contact page, and an about page with basic site features.

## Project Overview
The website is designed for a fictional store called ALIF.  
It contains:
- A hero section with welcome text
- A menu button
- Navigation bar with links
- Multiple sections with content and images
- A responsive layout

## Files Included
- index.html – Homepage of the website
- services.html – Services to order calligraphy
- contact.html – Contact page
- style.css – Main CSS file
- script.js – JavaScript file
- about.html- information of the page
- /images/ - calligraphy images

## Features
- Responsive navigation bar
- Background images
- Buttons and moving background

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Download or clone all project files.
2. Keep all files in the same folder structure.
3. Open home.html (or index.html) in any browser.
